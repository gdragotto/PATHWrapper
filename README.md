# PATHWrapper
A C++ Wrapper for the [PATH](http://pages.cs.wisc.edu/~ferris/path.html) Solver that manages Mixed Complementarity Problems (MCP), inspired from its [AMPL](https://github.com/ampl/pathlib) library.
